# 📝 Smart To-Do Manager

A simple and interactive task management application built with **Python** and **Streamlit**.

The app helps users organize their daily tasks, set priorities, track progress, and manage completed tasks through an easy-to-use interface.

---

## 🌐 Live Demo

Try the app here:

https://to-do-app-fu3lhu3ojgy8eq9tbgt6ab.streamlit.app/

---

## ✨ Features

- ✅ Add new tasks
- 🔍 Search tasks
- 🎯 Set task priorities (High, Medium, Low)
- 📅 Assign due dates
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🧹 Clear completed tasks
- 📊 Task statistics dashboard
- 📈 Progress tracker
- 💾 Automatic data saving using JSON

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/desire-orokumue/Todo-App.git
```

2. Open the project folder

```bash
cd To-Do-App
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
